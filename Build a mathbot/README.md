This workshop revolved around the various Math function in javascript from Math.floor() Rounds a number down to the nearest whole number. 
Math.floor(4.9);   // 4
Math.floor(4.1);   // 4
Math.floor(-4.1);  // -5
Math.ceil() Rounds a number up to the nearest whole number.
Math.ceil(4.1);   // 5
Math.ceil(4.9);   // 5
Math.ceil(-4.9);  // -4
Math.random() Generates a pseudo-random decimal number from 0 (inclusive) up to but not including 1.
Math.random();  // e.g., 0.347829
Math.max() Returns the largest number from a list.
Math.max(3, 10, 7);  // 10
Math.min() Returns the smallest number from a list.
Math.min(3, 10, 7);  // 3
Math.round() Rounds to the nearest whole number.
.5 and above → rounds up
Below .5 → rounds down
